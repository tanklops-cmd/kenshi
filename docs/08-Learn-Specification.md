# 08 - Learn Specification

## Purpose

Learn is the general skills library / kendo academy.

It answers:

> What do I want to understand?

## Learn Categories

The library is organised into twenty categories. Categories with zero articles remain
visible to communicate the full scope of the encyclopedia.

- Fundamentals
- Footwork
- Kamae
- Seme
- Maai
- Tenouchi
- Shikake-waza
- Oji-waza
- Hiki-waza
- Tsuki
- Kihon
- Drills
- Kata
- Shinpan
- Shiai
- Reiho
- Equipment
- Grading
- Terminology
- History

Each category maps to a `LearnCategory` enum value in the domain layer.
Article counts are displayed beside each category on the category list screen.

## Content Style

- Clear
- Respectful
- Non-dogmatic
- Avoid claiming to replace Sensei
- Use phrases like "common teaching points" where appropriate

## Relationship to Practice

Learn is objective reference.

Practice is personal development.

A Learn page can link to the user's related Practice Topic.
