# 11 - Search Specification

## Purpose

Search makes accumulated learning useful over time.

It should search:

- Practice Topics
- Reflections
- Guidance
- Moments
- Learn Library

## Philosophy

Every reflection should become easier to find than it was to write.

## Version 1 Requirements

- Local search
- Offline search
- Fast response
- Search results grouped by type

## Future

AI-enhanced semantic search may be considered later, but Version 1 should not require it.
