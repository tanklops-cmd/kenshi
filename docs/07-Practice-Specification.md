# 07 - Practice Specification

## Purpose

Practice is the user's personal kendo development area.

It answers:

> What am I working on?

## Practice Topics

There is one unified Practice Topic page for both Techniques and Fundamentals.

Examples:

### Techniques

- Men
- Kote
- Debana-men
- Kaeshi-do
- Hiki-men

### Fundamentals

- Seme
- Maai
- Kamae
- Tenouchi
- Footwork
- Zanshin
- Ki-Ken-Tai-Ichi

## Practice Topic Page Layout

Top to bottom:

1. Topic Header
2. Current State
3. Current Intention, if relevant
4. Mental Cues
5. Active Guidance
6. Moments
7. Reflection Timeline
8. Related Topics
9. Reference Summary
10. Link to full Learn guide

## Mental Cues

Mental cues exist at two levels:

- Global cues
- Topic-specific cues

## Personal First

Personal user content always appears before reference content.
