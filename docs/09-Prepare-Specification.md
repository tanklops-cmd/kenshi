# 09 - Prepare Specification

## Purpose

Prepare helps the user walk into the next keiko with one clear intention.

It answers:

> What should I remember before training?

## Layout

Top to bottom:

1. Next Keiko / Tonight
2. Current Intention
3. Global Mental Cues
4. Relevant topic-specific cues
5. Recent Guidance
6. Optional recent Moment
7. Quiet closing state, e.g. "Ready."

## Philosophy

Prepare should be calm and minimal.

It is not a dashboard.

It should be useful in 30 seconds or less.
