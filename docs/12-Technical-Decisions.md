# 12 - Technical Decisions

## Framework

Flutter.

Reason:

- Android support
- Windows support
- Future iOS/macOS support
- One codebase
- Good offline support

## Database

SQLite.

Reason:

- Offline-first
- Fast
- Reliable
- Good local ownership

## Authentication

No account required to start.

Optional login/sync may be added later.

## State Management

Recommended: Riverpod.

## Routing

Recommended: GoRouter.

## Development Environment

Visual Studio Code with Codex assistance.

## Project Management

GitHub repository, issues, and projects.
