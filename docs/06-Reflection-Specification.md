# 06 - Reflection Specification

## Purpose

Help the user capture learning from keiko in 2-5 minutes and leave with one clear intention for next keiko.

## Core Fields

### Session Context

Optional.

Examples:

- Club keiko
- Seminar
- Grading practice
- First session back
- Tired after work
- New shinai

### Guidance

Individual coaching notes.

Each guidance note should be separate.

Examples:

- Right hand too dominant
- Win centre before attacking
- Better zanshin

Guidance may later support attachments / Moments.

### Reflection

Large writing area for the user's own thoughts.

Suggested placeholder:

> What stood out from today's training?

### Next Keiko / Current Intention

Mandatory one-sentence field.

Prompt:

> What will you deliberately work on next time?

## Finish Action

Button text:

> Finish For Now

After finishing, show a quiet summary:

- Reflection saved
- Guidance count
- Current Intention
- Linked Practice Topics, if any

## Later Reflection

When returning to a previous session, the app should prompt:

> Anything else come to mind?

Later thoughts should be appended, not treated as overwrites.
