# 08 - Learn Specification

## Purpose

Learn is the general skills library / kendo academy.

It answers:

> What do I want to understand?

## Learn Categories

Version 1 may include placeholders for:

- Techniques
- Fundamentals
- Kata
- Training
- Equipment
- Rules
- Grading
- Etiquette

## Content Style

- Clear
- Respectful
- Non-dogmatic
- Avoid claiming to replace Sensei
- Use phrases like "common teaching points" where appropriate

## Relationship to Practice

Learn is objective reference.

Practice is personal development.

A Learn page can link to the user's related Practice Topic.
