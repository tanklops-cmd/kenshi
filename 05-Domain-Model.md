# 04 - Navigation

Version 1 navigation is fixed.

```text
Today
Reflect
Practice
Learn
Prepare
```

## Today

Answers: Where am I in my current training cycle?

Displays next keiko, current intention, recent reflection status, and quiet prompts such as "Anything else come to mind?"

## Reflect

Answers: What did I learn from training?

Used after keiko to capture guidance, reflection, and the next intention.

## Practice

Answers: What am I working on?

Contains the user's personal kendo development organised by Practice Topics.

## Learn

Answers: What do I want to understand?

A general skills library / kendo academy containing structured reference material.

## Prepare

Answers: What should I remember before the next keiko?

Shows current intention, mental cues, relevant guidance, and possibly one recent Moment.
